## A. 配色（八选一 or 逐项自定义）

> 二选一。方式①省事；方式②精确。颜色格式：`#` + 3/6/8 位十六进制（如 `#2563eb`）。

**方式① 选一套现成配色**（勾一个）：
[ ] 靛蓝　[ ] 海洋　[ ] 日落　[ ] 森林　[ ] 玫瑰　[ ] 石墨

**方式② 自定义 8 色**（只填想定的，留空用默认）：

| 角色 | 用在哪里 | 你的色值 |
| --- | --- | --- |
| 品牌主色 `accent` | 按钮 / 链接 / 高亮 / 当前态 | `__________` |
| 品牌色上文字 `accent_ink` | 实心按钮上的字（通常 `#ffffff`） | `__________` |
| 页面背景 `bg` | 全站底色（深色主题填深色值） | `__________` |
| 卡片背景 `surface` | 卡片 / 面板 | `__________` |
| 次级背景 `surface_2` | 输入框 / 弱化区块 | `__________` |
| 主文字 `text` | 标题 / 正文 | `__________` |
| 次级文字 `text_muted` | 说明 / 元信息 | `__________` |
| 边框 `border` | 分割线 / 描边 | `__________` |

> 想要「深色主题」？把 `bg`/`surface`/`surface_2` 填深色、`text` 填浅色即可。品牌色会同时用于浅色与深色两种模式。
> ⚠️ 请自查对比度：`text`↔`bg`、`accent_ink`↔`accent` 要清晰可读。
